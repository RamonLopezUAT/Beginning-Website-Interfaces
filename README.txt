Description:
This project is a styled version of Article I of the U.S. Constitution. The webpage uses CSS to create a visually engaging experience, including typography styling, background colors, and responsive design features. The page includes a navigation bar, headers, sectioned content, and a footer.

Files:
  - constitution.html: Contains the HTML structure and content for the webpage.
  - style.css: Contains the CSS styling rules for the webpage.

Features:
  - Font styling (Georgia for the body text).
  - Dark themed header and footer with contrasting white text.
  - Sectioned content with heading and paragraph styles.
  - Responsive design adjustments for smaller screen sizes.

Validation:
  - HTML validation: Passed with no errors (validated using https://validator.w3.org/).
  - CSS validation: Passed with no errors (validated using https://jigsaw.w3.org/css-validator/).
